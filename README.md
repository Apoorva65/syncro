# Syncro

Syncro is a simple project and task management web app built with React and Firebase. It allows users to create projects, manage tasks within each project, and track task completion with a clean and minimal UI.

## Features

- User authentication (Sign up / Login Logout)
- Create and view projects
- Add tasks under each project
- Mark tasks as completed
- Delete tasks with confirmation
- Task count per project
- Persistent data using Firebase Firestore
- Protected routes for authenticated users

## Tech Stack

- **Frontend:** React, React Router, Tailwind CSS
- **State Management:** Redux Toolkit
- **Backend/Auth:** Firebase Authentication
- **Database:** Firebase Firestore
